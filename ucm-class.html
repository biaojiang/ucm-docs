Unobserved components model:¶

\[\begin{split}y_t &= \mu_t + c_t\\ \mu_{t+1} &=\mu_t + \xi_t,\quad \xi_t \sim N(0, \sigma_\xi^2)\\ c_{t+1} &= \rho_c\left( \tilde{c}_t \cos\lambda_c +\tilde{c}_t^* \sin\lambda_c\right)+ {\tilde{\omega}}_t\\ c_{t+1}^* &= \rho_c\left( -{\tilde{c}}_t \sin\lambda_c +\tilde{c}_t^* \cos\lambda_c \right) + {\tilde{\omega}}_t^*\\ \tilde{\omega}_t,\;\tilde{\omega}_t^*& \sim \text{i.i.d. } N(0,\sigma_{\tilde{\omega}}^2)\end{split}\]
